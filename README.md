# TokyoDark.nvim

A dark Neovim theme written in Lua

## Installation

Using [lazy.nvim](https://github.com/folke/lazy.nvim)

```lua
{
  "username/nightfall.nvim",
  lazy = false,
  priority = 1000,
  opts = {},
}
```

## Usage

```lua
vim.cmd[[colorscheme nightfall]]
```

## Features

- Support for popular plugins
- Customizable
- Written in Lua
- Treesitter support
